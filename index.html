T4trace to neuron research

T4trace ("T" for trace) is designed to analyze large-scale neurite micropic images. To appliable on massive analysis, the application hide parameters or manual manipulations rather automated manner. Still, it is available on general desktop environment. With JRE7 and Equinox framework basis, most Java 1.7 runnable systems are supported.

If you haven't installed JRE7, try here(http://java.com/download) to get it

T4trace to head-first's

  1. Open T4trace application
  2. From Menu, T4trace > Set workspace, select folder contains target images
  3. Browse images from left panel, dbl-clk a image
  4. You'll see there.
OR,
  1. Open T4trace application
  2. Select workspace
  3. dbl-clk a folder contains images
  4. Check items (images) on right-hand-side list, and clk "START" button
  5. It takes some time, get a cup of coffee - I mean, instant.

!Caution: T4trace can takes 8-bit, TIF format images. Monochrome recommended, but you can select color filter on multiple analysis mode.

 

"T" for trace and Japanese term, "Takumi(匠)"

The name T4trace, you choose to read either [Tea-for-trace] or [Takumi-trace]. "Takumi"(匠) is Japanese term which means "Elaborated skill, or master of the skill". You may find it's "Takumi-ness" in short.
