Single mode

Grid size setting

For better performance, T4trace use initial node sampling grids. As grid size getting smaller, results are getting precised, but time-cost will be squared. So, if you try bigger (like, 1024x1024 px or bigger) image, put slide "grid size" on Main menu > T4trace > Preference 2 or 4 recommended.

Canvas Zoom

First, click canvas to get system focus. If you put CTRL+Arrow-UP will magnify the canvas (zoom-in). As same manner, CTRL+Arrow-Down is zoom-out. If you put CTRL+0 (Number zero) will restore default zoom scale. Zoom scalability is limited about 30~300% by origin.

Save current canvas to PNG image

Right click on the canvas will open popup menu. Select "Save current canvas to Image" on the menu. Set desired file path to put.

Save table to text file

Right click on the table will open popup menu. Select "Save to text" on the menu. Set path then check the result. Images are neglected.

Save layers to separated PNG images

Right click on the canvas, select "Selections to Image". This menu save selected layers into seperated PNG files, with transparency preserved. Enjoy on your paper.

Multi mode

Interactive filter

On the right-hand-side panel, you may see toggle button saying "Include", and text field next to it. Those are interactive filter. In case there are tons of images on the folder, it very hard to check items one-by-one.

So, let's say -- if you want to see "DIV3" results first. With button saying "Include", put "DIV3" on the text field. List items will be filtered that 'includes "DIV3" on its name'. If you want to run except, say ".x.filtered" files, Toggle left button to "Except", than put the term on the field.

Plus, the filter run based on Java Regular expression. That is, you can set regular expression directly into text filter. Try your own, for example, "DIV\d+_1_3.*$".

Biggest trick is, "Select all" button on the bottom of the list doesn't actually intended "ALL". It means, "All" items, which is shown on the list. And you can filter items interactively. Got the context here? Filter - Select all - Start. There you go.

Save to Files

Although, yes, multi-mode window table also support "Save to text" exactly as single mode. That's not all. Before start processing, try to sel;ect "FILEs To:" check box. It will open folder selection dialog. Select a folder you want to put results. If it set properly, you may see the folder name next to checkbox text. Then, run the process and check the folder. Enjoy!
